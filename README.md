## Project Overview
This project focuses on analyzing bank loan data to understand loan performance, customer behavior, and risk patterns. The dashboard helps stakeholders track key loan metrics and make data-driven decisions.

## Objectives
- Analyze loan distribution across customer segments
- Identify default and delinquency trends
- Track loan performance using KPIs
- Provide interactive visual insights for decision-making

## Tools & Technologies
- Excel
- Power BI
- Tableau
- SQL

## Key Features
- KPI metrics such as Total Loans, Total Loan Amount, Default Rate, and Delinquency Rate
- Loan analysis by region, branch, age group, and loan grade
- Interactive dashboards with filters and slicers
- Data cleaning and transformation using SQL

## Outcome
The dashboard provides a clear view of loan risk and performance, helping banks improve credit decisions and portfolio management.
